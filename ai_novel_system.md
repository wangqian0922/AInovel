# AI 小说生成器总体架构（当前稳定版）

## 1. 系统定位

这是一个"分层 prompt 编排的长期连载小说生成系统"，核心特征： - 无
agent - 无 tool call - 无自动状态写入 - 完全手动控制 - 强确定性生成

------------------------------------------------------------------------

## 2. Prompt 拼装顺序（严格固定）

每次生成输入 LLM 的内容顺序如下：

1.  Session
2.  System Insert Prompt
3.  WorldBook.fixed
4.  WorldBook.keyword
5.  StateSet
6.  History Buffer
7.  User Input

------------------------------------------------------------------------

## 3. 各模块定义

### 3.1 Session

-   Markdown / Prompt 文本
-   定义故事纲领
-   最高优先级约束
-   每次会话固定注入

------------------------------------------------------------------------

### 3.2 System Insert Prompt

-   static per-session
-   控制输出行为
    -   字数要求
    -   风格约束
    -   节奏要求
-   不参与世界设定

------------------------------------------------------------------------

### 3.3 WorldBook

-   世界知识库（百科系统）

结构： - fixed：不可变规则 - keyword：纯关键词命中触发（无语义匹配）

规则： - 仅关键词命中才注入 - 无 embedding / 无语义扩展

------------------------------------------------------------------------

### 3.4 StateSet

-   当前状态快照（唯一真相）
-   overwrite-only
-   不保留历史
-   完全由用户维护

------------------------------------------------------------------------

### 3.5 History Buffer

-   滑动窗口结构
-   默认 200（可配置）
-   FIFO 裁剪策略
-   支持：
    -   编辑 content
    -   hidden 标记（不参与上下文）
    -   id 固定

规则： - 不生成空 user 记录 - hidden 不参与 prompt - FIFO 删除最旧

------------------------------------------------------------------------

### 3.6 User Input

-   当前生成指令
-   可能包含剧情引导

------------------------------------------------------------------------

## 4. 优先级规则（冲突处理）

严格优先级：

Session \> System Insert Prompt \> WorldBook.fixed \> WorldBook.keyword
\> StateSet \> History Buffer \> User Input

冲突处理方式： - 以高优先级为准 - 不做自动修复 - 用户手动修改结构

------------------------------------------------------------------------

## 5. 系统行为原则

-   所有状态均由外部维护
-   LLM 只负责文本生成
-   不参与状态更新
-   不具备 agent 行为
-   不具备工具调用能力

------------------------------------------------------------------------

## 6. History Buffer 规则

-   最大长度可配置
-   FIFO 滑动窗口
-   entry 结构：
    -   id（稳定）
    -   role
    -   content
    -   hidden flag
-   hidden 不参与 prompt
-   支持编辑但不改变 id

------------------------------------------------------------------------

## 7. WorldBook 规则

-   strict keyword matching
-   无语义匹配
-   无模糊召回
-   fixed 优先于 keyword

------------------------------------------------------------------------

## 8. 系统本质

该系统是：

"确定性 prompt 拼装型长期连载小说生成器"
